Airgead Banking Investment Calculator

A C++ console application that calculates compound interest growth and provides yearly summaries based on user inputs. This artifact is used for the Algorithms and Data Structures category in CS-499.

Original Functionality

  • User input for initial investment, monthly deposit, interest rate, and investment duration
  • Compound interest calculations (monthly compounding)
  • Formatted terminal output for year-end balances and interest earned
  • Error handling and input validation

Enhancements for CS-499

  • Added monthly breakdown logic to increase accuracy
  • Replaced static arrays with std::vector of custom structs
  • Improved naming conventions and consistency
  • Modularized core logic to increase reusability
  • Refined input validation and error handling
  • Expanded comments and documentation for maintainability

Skills Demonstrated

  • Compound interest calculation using loops and formulas
  • Custom C++ structures and dynamic memory storage (std::vector)
  • Input validation and structured error messaging
  • Modular and reusable function design

Course Outcomes Met

  • Outcome 3: Design and evaluate computing solutions (algorithm correctness and performance)
  • Outcome 4: Use well-founded techniques and tools (C++ STL, user interface validation)
