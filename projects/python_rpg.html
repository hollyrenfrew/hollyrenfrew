Project Overview

A console-based RPG where the player navigates rooms (north, south, east, west) and chooses between melee and ranged weapons with different strength, accuracy, and critical chance values. The original game runs fully in memory and resets each session.

Planned Enhancement (per CS-499)

Enhance the project by integrating a persistent database system using SQLite. Player profiles, stats, chosen boons/banes, inventory items (such as arrows), room location, and combat history will be saved and restored across sessions. This turns the original prototype into a system with long-term progression and real saved game files.
