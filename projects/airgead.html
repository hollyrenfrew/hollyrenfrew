Project Overview

A C++ console program that computes compound interest and reports year-end balances. This artifact is used for the Algorithms & Data Structures category in CS-499.

Key Features

  • User-driven inputs: initial investment, monthly deposit, annual interest, years
  • Formatted annual report
  • Input validation and error handling

Planned Enhancement (per CS-499)

Upgrade to monthly compounding with annual aggregation and store results in a dynamic std::vector of structs (year, ending balance, interest earned). Improves accuracy, scalability, and data access patterns.

Skills Demonstrated

  • Algorithm refinement (iterative monthly compounding)
  • Data structure use (std::vector & custom record type)
  • Modular refactoring and readability
