Project Overview

The Airgead Banking App is a financial simulation tool that calculates investment growth over time. It allows users to enter savings goals, rates, and time frames to analyze outcomes.

Key Features

  • User-input driven financial calculations
  • Formatted investment reports
  • Data validation and input handling
  • Clear menu-driven interface

Planned Enhancements

  • Store and retrieve financial scenarios using a real database
  • Add data persistence and reporting
  • Improve input validation security

Skills Demonstrated

Data handling, C++ logic, input validation, financial computation algorithms, refactoring.
