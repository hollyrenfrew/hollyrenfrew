Project Overview

This project is a 3D recreation of my living room using modern OpenGL. It demonstrates 3D rendering, lighting, camera controls, textures, and scene management.

Key Features

  • Perspective camera movement
  • Dynamic lighting
  • 3D object modeling and placement
  • Shader-based rendering pipeline

Planned Enhancements

  • Add interactive object selection
  • Introduce basic physics (gravity and collision)
  • Improve structure and documentation

Skills Demonstrated

C++, OpenGL, shaders, event-driven programming, 3D math, computer graphics pipeline.
