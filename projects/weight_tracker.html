Project Overview

An Android app for tracking daily weights, setting a target, and receiving notifications when above/below the goal. Uses SQLite, RecyclerView, and a line chart for trends.

Key Features

  • Target weight with above/below toggle
  • Daily entries list (add/update/delete)
  • Trend chart (MPAndroidChart)
  • Optional SMS alert after consent

Planned Enhancement (per CS-499)

Focus on secure logins: replace plain SharedPreferences “remember me” with encrypted session storage (AndroidX EncryptedSharedPreferences), add rate-limiting/temporary lockout for repeated failures, optional 2FA, and generic error messages to reduce user enumeration.

Skills Demonstrated

  • Secure software design (encryption, session tokens, lockout)
  • Android platform APIs (SQLite, RecyclerView, Notifications)
  • Usability and validation
